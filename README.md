# Shaale-Vikas

## Android App Development using GenAI

Shaale-Vikas is an Android application developed to support rural school infrastructure development by connecting schools with alumni and donors. The application acts as a transparent platform where schools can post their needs and alumni can contribute through donations or support.

---

# Problem Statement

Many rural schools face infrastructure problems such as damaged classrooms, broken furniture, lack of study materials, and maintenance issues. Alumni are willing to help, but they often do not know the current needs of the school. This project helps bridge the communication gap between schools and alumni.

---

# Project Objective

* To connect rural schools and alumni through a digital platform.
* To allow schools to post their infrastructure and repair needs.
* To enable alumni and donors to contribute easily.
* To provide transparency in donations and school development activities.
* To display real-time progress updates using Firebase.

---

# Features

* User Login and Registration

---

# System Flow

1. User opens the application.
2. User logs in or registers.
3. Headmaster posts school needs.
4. Alumni view needs and donate.
5. Firebase updates data in real-time.
6. Progress and impact are displayed to users.
